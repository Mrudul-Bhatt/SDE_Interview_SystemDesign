## 2. Write Path — Create Short URL

```mermaid
sequenceDiagram
    participant C as Client
    participant API as Write Server
    participant V as Validator
    participant IDG as Redis Counter
    participant DB as PostgreSQL Primary
    participant CACHE as Redis Cache

    C->>+API: POST /shorten<br/>{url, alias?, ttlDays?}

    API->>+V: Validate URL format + SafeBrowsing check
    V-->>-API: OK / REJECTED

    alt Custom alias provided
        API->>+DB: SELECT WHERE short_code = alias
        DB-->>-API: exists?
        alt Alias taken
            API-->>C: 409 Conflict
        end
    else Auto-generate code
        API->>+IDG: INCR url_id
        IDG-->>-API: id = 100042
        API->>API: Base62Encode(100042) = "q0G"
    end

    API->>+DB: INSERT (short_code, long_url, expires_at, user_id)
    DB-->>-API: OK

    API->>CACHE: SET q0G → long_url EX ttl_seconds
    Note over API,CACHE: Pre-warm cache on creation

    API-->>-C: 201 Created<br/>{shortUrl: "sho.rt/q0G", expiresAt: ...}
```

## 3. Read Path — Redirect

```mermaid
sequenceDiagram
    participant C as Client
    participant CDN as CDN Edge
    participant API as Read Server
    participant CACHE as Redis Cache
    participant DB as PostgreSQL Replica
    participant MQ as Kafka (async)

    C->>+CDN: GET sho.rt/q0G

    alt 301 cached at CDN
        CDN-->>C: 301 Redirect → long_url
        Note over CDN,C: No server hit — instant
    else Cache miss at CDN
        CDN->>+API: GET /q0G

        API->>+CACHE: GET q0G
        alt Cache HIT (~99% of traffic)
            CACHE-->>API: {longUrl, expiresAt}
            API->>API: Check expiry
            alt Not expired
                API-->>CDN: 302 Redirect → long_url
                CDN-->>-C: 302 Redirect
                API-)MQ: Publish click event (async, non-blocking)
            else Expired
                API-->>C: 410 Gone
            end
        else Cache MISS (~1% of traffic)
            CACHE-->>-API: null
            API->>+DB: SELECT long_url, expires_at, is_active<br/>WHERE short_code = 'q0G'
            DB-->>-API: UrlRecord (or null)
            alt Not found
                API-->>-C: 404 Not Found
            else Found
                API->>CACHE: SET q0G → longUrl EX min(24h, remaining_ttl)
                API-->>CDN: 302 Redirect
                CDN-->>C: 302 Redirect
                API-)MQ: Publish click event (async)
            end
        end
    end
```

## 7. Cache Flow Decision Tree

```
GET /:code
    │
    ▼
Redis.GET(code)
    │
    ├── HIT ──► check expiry ──► expired? ──► 410 Gone
    │                    │
    │                    └──► valid ──► 302 Redirect ──► async log click
    │
    └── MISS
            │
            ▼
        DB.SELECT(code)
            │
            ├── NOT FOUND ──► 404
            │
            ├── FOUND + expired ──► 410 Gone
            │
            └── FOUND + valid
                    │
                    ▼
                Redis.SET(code, url, EX = min(86400, remaining_seconds))
                    │
                    ▼
                302 Redirect ──► async log click
```
